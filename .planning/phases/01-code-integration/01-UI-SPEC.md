# Phase 1 — UI Design Contract

## Spacing Scale

Declared values (multiples of 4, from `css/variables.css` `--space-*` tokens):

| Token | Value | CSS Variable | Usage |
|-------|-------|-------------|-------|
| xs | 4px | `--space-xs` | Icon gaps, inline padding |
| sm | 8px | `--space-sm` | Compact element spacing |
| md | 16px | `--space-md` | Default element spacing |
| lg | 24px | `--space-lg` | Section padding, content-body padding |
| xl | 32px | `--space-xl` | Layout gaps, page-header padding |
| 2xl | 48px | `--space-2xl` | Major section breaks |

Additional spacing used in layout but not tokenized:
- 20px -- content-header padding, content-body padding
- 28px -- page-header padding
- 64px -- sidebar width (`--sidebar-w: 264px`, not a spacing token)

Exceptions: Icon-only buttons use 6px padding (`icon-only-btn`, `acc-btn-icon`); toolbar buttons use 32px height.

**Source:** Extracted from `css/variables.css` lines 62-67, `css/components/button.css` lines 61-74, 105, `css/layout.css` lines 283, 336, 371.

## Color

| Role | Value | CSS Variable | Usage |
|------|-------|-------------|-------|
| Dominant (60%) | `#f0f2f5` | `--bg-body` | Page background, main content area |
| Secondary (30%) | `#ffffff` | `--bg-card`, `--bg-elevated` | Cards, elevated surfaces, content panels |
| Accent (10%) | `#6366f1` | `--primary` | Primary buttons, active nav item indicator, page header gradient, CTA actions |
| Destructive | `#ef4444` | `--danger` | Delete buttons, danger actions, error states |

Semantic colors (reserved for status display):
- `--success: #10b981` -- success states, enabled badges
- `--warning: #f59e0b` -- warning states, pro badge
- `--info: #06b6d4` -- informational states

Dark surface:
- `--bg-sidebar: #1e1b4b` -- sidebar background with gradient overlay

Text hierarchy:
- `--text-main: #111827` -- primary body text
- `--text-secondary: #374151` -- secondary text, descriptions
- `--text-muted: #6b7280` -- placeholders, helper text
- `--text-on-dark: #e0e7ff` -- text on dark sidebar

Accent reserved for:
- Primary action buttons (`.btn-primary`, `.compact-btn-primary`)
- Active navigation item indicator (`.nav-item.active::before`)
- Page header gradient background
- Selected/active state indicators
- Link hover states
- Focus rings
