<objective>
Create a Pool-to-Channel bridge that syncs available pool accounts to the gateway channels table. This allows the existing gateway failover (3 retries) to work with pool accounts without modifying the pipeline itself.
</objective>

<tasks>

<task type="auto">
  <name>Task 1: Create PoolChannelBridge service</name>
  <files>src/services/poolChannelBridge.js</files>
  <action>
    Create src/services/poolChannelBridge.js that syncs pool accounts to gateway channels.

    The bridge:
    1. Reads all available pool accounts with credentials
    2. For each account, creates or updates a channel in the channels table
    3. Channel name: `[Pool] email` or `[Pool] displayName`
    4. Channel type: maps pool provider_type to channel type (windsurf→codex, openai→openai, etc.)
    5. Channel ordering_weight: based on health_score (higher = preferred)
    6. Channel status: follows pool account status (available→enabled, disabled→disabled)
    7. Marks channels as deleted when pool account is soft-deleted

    Constructor: (getDb) - receives the gateway database getter
    Method: sync(poolAccounts) - takes array of pool accounts, syncs to channels table
    Method: removeOrphaned() - removes pool channels that no longer exist in pool
  </action>
</task>

<task type="auto">
  <name>Task 2: Add pool-sync-channels IPC handler</name>
  <files>src/main/ipc/pool.js</files>
  <action>
    Add IPC handler in pool.js:
    ```javascript
    ipcMain.handle('pool-sync-channels', async (event) => {
      try {
        const { poolService, getGatewayDb } = deps;
        if (!getGatewayDb) return { success: false, error: 'Gateway DB not available' };
        const PoolChannelBridge = require('../../services/poolChannelBridge');
        const bridge = new PoolChannelBridge(getGatewayDb);
        const accounts = poolService.getAll({ status: 'available' });
        bridge.sync(accounts);
        bridge.removeOrphaned();
        return { success: true, synced: accounts.length };
      } catch (err) {
        return { success: false, error: err.message };
      }
    });
    ```
  </action>
</task>
